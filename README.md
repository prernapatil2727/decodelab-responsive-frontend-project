## 📌 About the Project

CareerPilot AI is a responsive front-end web application developed using **HTML, CSS, and JavaScript**.

The platform helps students discover various technology careers, explore learning roadmaps, compare salary insights, and plan their career journey through an interactive and user-friendly interface.

This project was created as part of my **DecodeLabs Full stack Development Internship**.
